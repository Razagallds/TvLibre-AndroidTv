# TV Libre

App de TV en vivo para Android TV construida con Expo.

## Requisitos

- Node.js 18+
- Expo CLI

## Instalación

```bash
npm install
```

## Ejecutar

```bash
npx expo start
```

## Estructura

- `App.js` - Componente principal
- `index.js` - Entry point
- `android/` - Build Android nativo
